# 🚀 Articles (完成品プロンプト集)

ここでは、Geminiと共に磨き上げ、実際にnoteで公開した完成版のプロンプトを管理しています。

## 📁 カテゴリ別リスト
※プロンプトが増えてきたら、ここにリンクを追記していきます。

- **[カテゴリ名]**
  - [プロンプト名](./ファイル名.md) : [短い説明文]
- **[カテゴリ名]**
  - [プロンプト名](./ファイル名.md) : [短い説明文]

---

## 📝 このフォルダの使い方
1. 各Markdownファイルを開き、冒頭の**「完成版プロンプト」**をコピーします。
2. 過去の試行錯誤（改善履歴）に興味がある場合は、ページ下部の**「改善履歴と過去のプロンプト」**を参照してください。
3. 活用事例や詳細なコツは、各ファイルに記載されている **note記事リンク** から確認できます。
