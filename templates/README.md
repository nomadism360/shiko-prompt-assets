# 📄 Templates

このフォルダには、プロンプト資産を管理するための共通フォーマットを保管しています。

## 🛠 使い方
1. `prompt-article-template.md` をコピーします。
2. `articles/` フォルダ内に新しいファイル（例：`sns-copy-v1.md`）を作成し、貼り付けます。
3. 項目に従って、Geminiとの対話ログや最新プロンプトを記入してください。

## 📌 執筆ルール
- **ファイル名**: `カテゴリ-タイトル.md`（すべて小文字、スペースはハイフン `-` に置換）
- **画像**: 画像を使用する場合は `assets/` に保存し、相対パスでリンクしてください。
